# 01 Course Overview

이 폴더는 `01`부터 `08`까지 이어지는 전체 과정의 큰 그림을 보는 곳입니다.

설치 방법이나 실행 명령을 자세히 설명하는 폴더가 아니라, **각 과정이 왜 필요한지, 어떤 순서로 연결되는지, 다음 과정으로 넘어가기 전에 무엇을 할 수 있어야 하는지**를 확인하는 안내 폴더입니다.

## 문서 구성

| 문서 | 내용 |
| --- | --- |
| [course-sequence.md](./course-sequence.md) | 01~08 과정 순서와 큰 흐름 |
| [course-map.md](./course-map.md) | 과정별 필수 목표, 선택/확장, 다음 과정 진입 기준 |

## 처음 보는 방법

1. [course-sequence.md](./course-sequence.md)에서 전체 흐름을 먼저 봅니다.
2. [course-map.md](./course-map.md)에서 각 과정의 필수 범위와 선택 범위를 구분합니다.
3. 실제 수업을 시작할 때는 각 과정 폴더의 `README.md`와 `SETUP.md`를 봅니다.
4. 설치가 필요하면 [02_setup-guides](../02_setup-guides/README.md)를 확인합니다.
5. 막히면 [03_learning-support](../03_learning-support/README.md)를 확인합니다.

## 과정 구분

| 구분 | 과정 | 성격 |
| --- | --- | --- |
| 기초 준비 | `01` | Python, Git/GitHub, VS Code 사용 흐름 |
| AI 웹 서비스 | `02`~`04` | FastAPI, Supabase, Streamlit, 실시간 로그 대시보드 |
| 단일 Agent | `05`~`06` | Prompt, Tool, RAG, Memory, LangGraph, 일정 조정 Agent |
| 멀티 Agent와 운영 | `07`~`08` | Multi-Agent, Docker Compose, GitHub Actions, AWS, Auto Healing |

`04`, `06`, `08`은 단위 프로젝트 과정입니다. 이 과정에서는 코드 구현뿐 아니라 설계 문서, 결과 보고서, 체크리스트까지 함께 정리합니다.
