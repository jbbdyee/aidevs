# 03. Beginner Error Guide

## SyntaxError

문법이 틀렸을 때 발생합니다.

예시:

```python
print("hello"
```

## NameError

정의하지 않은 이름을 사용했을 때 발생합니다.

예시:

```python
print(user_name)
```

## TypeError

자료형이 맞지 않을 때 발생합니다.

예시:

```python
print("나이: " + 20)
```

## IndentationError

들여쓰기가 잘못되었을 때 발생합니다.

예시:

```python
if True:
print("hello")
```
