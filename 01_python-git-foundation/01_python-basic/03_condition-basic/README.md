# 03_condition-basic

조건에 따라 다른 코드를 실행하는 방법을 학습합니다.

## 핵심 내용

```text
if
elif
else
match-case
비교 연산자
논리 연산자
```

## 예제 순서

| 파일 | 내용 |
| --- | --- |
| `01_if_only.py` | `if`만 사용해 입력한 숫자가 0보다 작으면 프로그램을 종료하고, 0 이상이면 계속 진행합니다. |
| `02_if_else.py` | `if`, `else`를 사용해 두 가지 경우 중 하나를 선택합니다. |
| `03_if_elif_else.py` | `if`, `elif`, `else`를 사용해 여러 조건 중 하나를 선택합니다. |
| `04_match_case_switch.py` | Python의 `match-case` 문법으로 switch 방식의 분기 처리를 연습합니다. |

## 실행

```powershell
python .\03_condition-basic\01_if_only.py
python .\03_condition-basic\02_if_else.py
python .\03_condition-basic\03_if_elif_else.py
python .\03_condition-basic\04_match_case_switch.py
```
