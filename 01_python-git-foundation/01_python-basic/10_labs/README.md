# 10_labs

`01_python-basic` 단원별 실습 모음입니다.

이 폴더의 Lab은 각 예제 파일을 그대로 따라 치는 단계가 아니라, 예제에서 배운 문법을 직접 조합해 작은 프로그램을 만들어 보는 단계입니다.

## 실행 기준

모든 실습은 `01_python-git-foundation` 최상위 폴더에서 실행합니다.

```powershell
cd C:\aidev\01_python-git-foundation
.\.venv\Scripts\Activate.ps1
```

실습 파일을 직접 만들 때는 아래처럼 `10_labs\practice` 폴더를 만들어 사용해도 됩니다.

```powershell
mkdir .\01_python-basic\10_labs\practice
```

## 완료 기준

각 Lab은 아래 기준으로 마무리합니다.

```text
1. 실행 명령을 기록했는가?
2. 정상 실행 결과를 확인했는가?
3. 오류가 있었다면 원인과 해결 방법을 적었는가?
4. 코드에 필요한 주석을 추가했는가?
5. 배운 문법이 어디에 쓰였는지 설명할 수 있는가?
```
