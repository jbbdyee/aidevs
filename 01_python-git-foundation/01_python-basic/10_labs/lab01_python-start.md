# Lab 01 - Python Start

출력, 주석, 입력, 실행 순서를 확인하는 실습입니다.

## 목표

- `print()`로 값을 출력할 수 있습니다.
- 주석을 사용해 코드 설명을 남길 수 있습니다.
- `input()`으로 값을 입력받을 수 있습니다.
- 코드가 위에서 아래로 실행된다는 흐름을 설명할 수 있습니다.

## 실습 1. 자기소개 출력

`practice\lab01_intro.py` 파일을 만들고 아래 조건을 만족하도록 작성합니다.

요구사항:

```text
1. 이름을 변수에 저장합니다.
2. 관심 있는 기술을 변수에 저장합니다.
3. print()로 두 값을 출력합니다.
4. 코드 중간에 주석을 2개 이상 작성합니다.
```

예상 출력 예:

```text
이름: Jean
관심 기술: FastAPI
```

## 실습 2. input 사용

`practice\lab01_input_intro.py` 파일을 만듭니다.

요구사항:

```text
1. input()으로 이름을 입력받습니다.
2. input()으로 오늘 학습할 내용을 입력받습니다.
3. 입력받은 값을 사용해 안내 문장을 출력합니다.
```

예상 실행 흐름:

```text
이름을 입력하세요: Jean
오늘 학습할 내용을 입력하세요: Python 함수
Jean님은 오늘 Python 함수 내용을 학습합니다.
```
