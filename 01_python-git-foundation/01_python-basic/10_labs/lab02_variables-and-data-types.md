# Lab 02 - Variables And Data Types

변수와 자료형을 확인하는 실습입니다.

## 목표

- 값을 변수에 저장할 수 있습니다.
- `int`, `float`, `str`, `bool`을 구분할 수 있습니다.
- `type()`으로 자료형을 확인할 수 있습니다.
- f-string으로 문자열을 만들 수 있습니다.

## 실습 1. 학습자 정보 만들기

`practice\lab02_profile.py` 파일을 만듭니다.

요구사항:

```text
1. 이름은 문자열로 저장합니다.
2. 나이는 정수로 저장합니다.
3. 평균 학습 시간은 실수로 저장합니다.
4. Python 설치 여부는 bool로 저장합니다.
5. 각 변수의 값과 type() 결과를 출력합니다.
```

## 실습 2. 간단한 점수 계산

`practice\lab02_score.py` 파일을 만듭니다.

요구사항:

```text
1. 국어, 영어, 수학 점수를 변수에 저장합니다.
2. 총점을 계산합니다.
3. 평균을 계산합니다.
4. f-string으로 결과를 출력합니다.
```

예상 출력 예:

```text
총점은 255점입니다.
평균은 85.0점입니다.
```
