# Lab 03 - Condition Basic

조건문을 사용해 프로그램 흐름을 나누는 실습입니다.

## 목표

- `if`만 사용하는 흐름을 만들 수 있습니다.
- `if/else`로 두 가지 경우를 나눌 수 있습니다.
- `if/elif/else`로 여러 조건을 처리할 수 있습니다.
- `match-case`로 메뉴 선택 구조를 만들 수 있습니다.

## 실습 1. 숫자 검사

`practice\lab03_number_check.py` 파일을 만듭니다.

요구사항:

```text
1. 숫자를 하나 입력받습니다.
2. 0보다 작으면 "음수입니다."를 출력하고 종료합니다.
3. 0이면 "0입니다."를 출력합니다.
4. 0보다 크면 "양수입니다."를 출력합니다.
```

## 실습 2. 점수 등급 계산

`practice\lab03_grade.py` 파일을 만듭니다.

요구사항:

```text
1. 점수를 입력받습니다.
2. 90 이상이면 A를 출력합니다.
3. 80 이상이면 B를 출력합니다.
4. 70 이상이면 C를 출력합니다.
5. 그 외에는 D를 출력합니다.
```

## 실습 3. match-case 메뉴

`practice\lab03_menu.py` 파일을 만듭니다.

요구사항:

```text
1. 메뉴 번호를 입력받습니다.
2. "1"이면 "사용자 조회"를 출력합니다.
3. "2"이면 "대화 저장"을 출력합니다.
4. "3"이면 "로그 확인"을 출력합니다.
5. 그 외에는 "알 수 없는 메뉴"를 출력합니다.
```
