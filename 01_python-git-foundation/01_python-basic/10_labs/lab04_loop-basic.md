# Lab 04 - Loop Basic

반복문을 사용해 같은 작업을 여러 번 실행하는 실습입니다.

## 목표

- `for`로 정해진 범위를 반복할 수 있습니다.
- `while`로 조건이 참인 동안 반복할 수 있습니다.
- `continue`와 `break`의 차이를 설명할 수 있습니다.
- `q`를 입력하면 종료되는 반복 프로그램을 만들 수 있습니다.

## 실습 1. for 합계 계산

`practice\lab04_for_sum.py` 파일을 만듭니다.

요구사항:

```text
1. 1부터 100까지 더합니다.
2. 최종 합계를 출력합니다.
3. range()를 사용합니다.
```

## 실습 2. continue와 break

`practice\lab04_continue_break.py` 파일을 만듭니다.

요구사항:

```text
1. 1부터 20까지 반복합니다.
2. 3의 배수는 출력하지 않고 건너뜁니다.
3. 숫자가 15가 되면 반복문을 종료합니다.
```

## 실습 3. q로 종료되는 입력 반복

`practice\lab04_until_q.py` 파일을 만듭니다.

요구사항:

```text
1. while True를 사용합니다.
2. 메시지를 계속 입력받습니다.
3. q를 입력하면 종료합니다.
4. q가 아니면 입력한 메시지를 다시 출력합니다.
```

## 실습 4. 반복 계산기 확장

`practice\lab04_calculator.py` 파일을 만듭니다.

요구사항:

```text
1. 첫 번째 숫자를 입력받습니다.
2. 두 번째 숫자를 입력받습니다.
3. 연산자 +, -, *, / 중 하나를 입력받습니다.
4. 계산 결과를 출력합니다.
5. 첫 번째 숫자 입력 위치에서 q를 입력하면 종료합니다.
```

## 확인 질문

```text
2. break는 반복문을 끝내나요?
3. while True를 사용할 때 반드시 필요한 것은 무엇인가요?
```
