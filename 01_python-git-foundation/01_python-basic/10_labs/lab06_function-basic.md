# Lab 06 - Function Basic

함수로 코드를 나누고 재사용하는 실습입니다.

## 목표

- 함수를 정의하고 호출할 수 있습니다.
- 매개변수와 반환값을 사용할 수 있습니다.
- 자료구조를 함수에 전달할 수 있습니다.
- 계산 로직을 함수로 분리할 수 있습니다.

## 실습 1. 인사 함수

`practice\lab06_greet.py` 파일을 만듭니다.

요구사항:

```text
1. 이름을 매개변수로 받는 greet 함수를 만듭니다.
2. 인사 문장을 return합니다.
3. 서로 다른 이름으로 2번 이상 호출합니다.
```

## 실습 2. 등급 함수

`practice\lab06_grade_function.py` 파일을 만듭니다.

요구사항:

```text
1. 점수를 매개변수로 받는 get_grade 함수를 만듭니다.
2. A, B, C, D 중 하나를 return합니다.
3. 여러 점수로 함수를 호출해 결과를 확인합니다.
```

## 실습 3. 평균 계산 함수

`practice\lab06_average.py` 파일을 만듭니다.

요구사항:

```text
1. 점수 list를 매개변수로 받습니다.
2. 평균을 계산해 return합니다.
3. 반환값을 출력합니다.
```

## 실습 4. 계산기 함수

`practice\lab06_calculator_function.py` 파일을 만듭니다.

요구사항:

```text
1. a, b, operator를 매개변수로 받는 calculate 함수를 만듭니다.
2. +, -, *, /를 처리합니다.
3. 0으로 나누는 경우 안내 문장을 return합니다.
4. 지원하지 않는 연산자도 안내 문장을 return합니다.
```

```text
2. 매개변수를 사용하는 이유는 무엇인가요?
3. 함수를 만들면 코드가 왜 읽기 쉬워지나요?
```
