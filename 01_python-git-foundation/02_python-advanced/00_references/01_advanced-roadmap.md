# 01. Advanced Roadmap

## 학습 흐름

```text
함수 심화
-> 모듈과 패키지
-> 예외 처리
-> 클래스와 객체 맛보기
-> 컴프리헨션 데이터 변환
-> JSON/날짜/경로 처리
-> 테스트
-> 프로젝트 구조
```

## 목표

이 과정을 마치면 다음을 할 수 있어야 합니다.

```text
함수로 코드를 나누기
모듈을 만들어 import하기
예외가 발생해도 안전하게 처리하기
클래스로 데이터와 동작 묶기
JSON 데이터 저장하고 읽기
list/dict 데이터를 필요한 형태로 변환하기
테스트 코드 작성하기
작은 프로젝트 구조 만들기
```
