# 2일 진행 로드맵

`02_python-advanced`는 Python 고급 문법 전체가 아니라 다음 과정으로 넘어가기 위한 연결 단원입니다.

## 1일차

```text
함수로 코드 나누기
dict 데이터를 함수에 전달하기
직접 만든 .py 파일 import하기
try/except로 오류 처리하기
raise로 검증 오류 만들기
JSON 파일을 안전하게 읽기
```

목표:

```text
한 파일에 긴 코드를 모두 쓰지 않고 함수와 파일로 나눌 수 있습니다.
오류 메시지에서 파일명, 줄 번호, 오류 종류를 찾을 수 있습니다.
```

## 2일차

```text
print와 assert 차이 이해하기
pytest로 테스트 실행하기
main.py와 app 폴더 구조 이해하기
JSON 저장/읽기 흐름을 프로젝트 구조에 넣기
dict와 dataclass object 차이 맛보기
```

목표:

```text
작은 함수가 예상대로 동작하는지 테스트할 수 있습니다.
FastAPI 프로젝트로 넘어가기 전에 main/services/storage/tests 구조를 이해할 수 있습니다.
```
