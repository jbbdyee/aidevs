# 10_labs

`02_python-advanced` 과정의 단원별 실습 모음입니다.

각 Lab은 예제 파일을 먼저 실행한 뒤, 직접 코드를 작성해보는 방식으로 진행합니다. 먼저 단원 폴더의 README와 예제 파일을 확인한 다음 Lab을 진행하면 흐름을 이해하기 쉽습니다.

## 실습 목록

```text
lab-01_function-advanced.md
lab-02_module-package-venv.md
lab-03_exception.md
lab-04_oop.md
lab-05_comprehension-iterator.md
lab-06_data-processing-advanced.md
lab-08_testing-code-quality.md
lab-09_project-structure.md
```

## 진행 순서

```text
1. 예제 파일 실행
2. 코드 주석 읽기
3. 값이나 문장을 바꿔보기
4. Lab 요구사항 구현
5. 실행 결과 확인
6. 막힌 부분을 README에 메모
```

## 진행 기준

```text
각 Lab은 해당 번호의 단원 README와 예제 파일을 먼저 확인한 뒤 진행합니다.
예를 들어 lab-05_comprehension-iterator.md는 05_comprehension-iterator 단원과 함께 봅니다.
실습 파일은 별도 .venv를 만들지 않고 01_python-git-foundation의 공통 .venv에서 실행합니다.
```
