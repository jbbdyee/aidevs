`01_streamlit-basic`을 진행하기 전에 읽는 참고 자료입니다.

이 폴더는 Streamlit을 처음 사용할 때 자주 헷갈리는 개념을 정리합니다. 예제 파일을 실행하기 전에 한 번 읽고, 실습 중 오류가 나면 다시 확인합니다.

## 문서 목록

```text
streamlit-basic-notes.md
```

## 먼저 확인할 내용

- Streamlit은 Python 코드로 웹 화면을 빠르게 만들 수 있는 도구입니다.
- `streamlit run 파일명.py` 명령으로 앱을 실행합니다.
- Streamlit 코드는 위에서 아래로 다시 실행되며, 화면 출력 결과는 브라우저에서 확인합니다.
- 입력 컴포넌트의 결과는 Python 변수에 저장됩니다.
- 처음에는 API나 데이터베이스보다 화면 출력, 입력, 레이아웃 흐름을 먼저 익힙니다.

```text

st.write와 print의 차이는 무엇인가요?
```
