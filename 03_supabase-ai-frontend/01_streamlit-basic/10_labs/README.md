# 10_labs

수업 중 함께 진행하는 실습 파일입니다.

이 실습은 Streamlit 기본 문법을 “작은 화면 하나”로 묶어 보는 단계입니다. 예제 파일을 그대로 실행한 뒤, 입력값과 출력 문구를 조금씩 바꾸면서 화면이 어떻게 변하는지 확인합니다.

## 실습 목록

```text
lab-01-profile-card-app.py
lab-02-learning-check-app.py
lab-03-simple-dashboard.py
```

## 실습 진행 방식

1. 예제 파일을 먼저 실행합니다.
2. 입력값을 바꾸며 화면 변화를 확인합니다.
3. 주석을 읽고 코드 흐름을 설명합니다.
4. 요구사항에 맞게 일부 코드를 직접 수정합니다.
5. 화면에서 입력 영역과 결과 영역이 구분되는지 확인합니다.

## 실행 예시

```powershell
cd C:\aidev\03_supabase-ai-frontend
.\.venv\Scripts\Activate.ps1
streamlit run .\01_streamlit-basic\10_labs\lab-01-profile-card-app.py
```

## 실습별 목표

| 실습 | 목표 |
| --- | --- |
| `lab-01-profile-card-app.py` | 텍스트 입력과 선택 입력을 사용해 간단한 프로필 화면을 만듭니다. |
| `lab-02-learning-check-app.py` | 숫자 입력과 조건문을 사용해 학습 상태 메시지를 표시합니다. |
| `lab-03-simple-dashboard.py` | sidebar, columns, metric을 조합해 기본 대시보드 화면을 구성합니다. |

## 완료 기준

- Streamlit 앱이 오류 없이 실행됩니다.
- 입력값을 바꾸면 화면의 결과가 함께 바뀝니다.
- 코드에서 입력, 처리, 출력 영역을 구분해 설명할 수 있습니다.
- 최소 한 줄 이상 직접 수정한 뒤 화면 변화를 확인합니다.
