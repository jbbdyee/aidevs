# 00_references

`02_streamlit-ui-components`를 진행하기 전에 읽는 참고 자료입니다.

이 폴더는 Streamlit UI 컴포넌트와 pandas DataFrame을 화면 구성 관점에서 정리합니다. 컴포넌트 이름을 외우기보다, 어떤 상황에서 어떤 UI를 선택하면 좋은지 판단하는 기준을 확인합니다.

## 문서 목록

```text
streamlit-ui-components-notes.md
```

## 먼저 확인할 내용

- 버튼은 사용자의 명시적인 실행 행동을 받을 때 사용합니다.
- 체크박스와 슬라이더는 설정값을 받거나 필터 조건을 만들 때 사용합니다.
- 폼은 여러 입력값을 한 번에 제출할 때 사용합니다.
- DataFrame은 표와 차트에 사용할 데이터를 정리하는 기본 형태입니다.
- 파일 업로드는 CSV 같은 외부 데이터를 화면에서 확인하는 첫 단계입니다.
