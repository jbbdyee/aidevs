# 20_assignments

수업 후 제출 과제 안내입니다.

과제는 Streamlit UI 컴포넌트를 조합해 작은 입력 화면이나 대시보드 화면을 만드는 방식으로 진행합니다. 복잡한 기능보다 입력, 처리, 출력이 명확하게 연결되는지가 중요합니다.

## 제출 과제

```text
assignment-01-survey-ui.md
assignment-02-table-chart-dashboard.md
```

## 제출 기준

- 버튼, 폼, 체크박스, 슬라이더 중 최소 2개 이상을 사용합니다.
- pandas DataFrame을 사용해 표를 출력합니다.
- 차트 또는 파일 업로드 기능 중 하나 이상을 포함합니다.
- 코드 주요 줄에 본인이 이해한 한글 주석을 작성합니다.
- `streamlit run` 명령으로 실행 가능해야 합니다.

## 권장 진행 순서

```text
1. 만들 화면의 목적을 한 문장으로 정합니다.
2. 어떤 입력 컴포넌트가 필요한지 고릅니다.
3. 입력값을 변수에 저장합니다.
4. 필요한 경우 DataFrame으로 데이터를 정리합니다.
5. 표, 차트, metric 중 어떤 방식으로 결과를 보여 줄지 정합니다.
6. streamlit run으로 실행하고 화면 흐름을 확인합니다.
```

## 평가 기준

| 항목 | 확인 기준 |
| --- | --- |
| 실행 가능성 | `streamlit run` 명령으로 오류 없이 실행됩니다. |
| UI 컴포넌트 사용 | 버튼, 폼, 체크박스, 슬라이더 등 입력 컴포넌트가 적절히 사용됩니다. |
| 데이터 표현 | DataFrame, 표, 차트 중 필요한 표현이 포함됩니다. |
| 화면 구성 | 입력 영역과 결과 영역이 구분되어 있습니다. |
| 코드 이해 | 주요 코드에 직접 이해한 주석이 들어 있습니다. |
