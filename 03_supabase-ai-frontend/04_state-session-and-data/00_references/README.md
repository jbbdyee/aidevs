## 샘플 백엔드 위치

실행 가능한 FastAPI 샘플 백엔드는 참고 문서 폴더가 아니라 아래 폴더에 따로 둡니다.

```text
C:\aidev\03_supabase-ai-frontend\04_state-session-and-data\00_sample_backend
```

샘플 계정은 다음과 같습니다.

```text
student / 1234
```

이 샘플은 Supabase를 대체하지 않습니다. 회원가입, token 저장, Authorization header, 사용자별 대화 이력 화면 흐름을 빠르게 확인하기 위한 보조 자료입니다.
