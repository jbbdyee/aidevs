# Assignment 01: Login State App

## 목표

Streamlit에서 로그인 상태를 관리하는 앱을 만듭니다.

## 요구사항

- 로그인 폼을 만듭니다.
- 로그인 성공 후 access token을 `st.session_state`에 저장합니다.
- 로그인 상태와 로그아웃 상태의 화면을 다르게 표시합니다.
- 로그아웃 버튼으로 토큰을 제거합니다.
- token 값은 화면에 그대로 노출하지 않고, 로그인 여부만 표시합니다.
- 백엔드가 준비되지 않은 경우 `00_sample_backend/main.py`를 사용할 수 있습니다.
