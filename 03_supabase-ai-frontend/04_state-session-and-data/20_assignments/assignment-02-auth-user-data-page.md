# Assignment 02: Auth User Data Page

## 목표

인증 토큰을 사용해 사용자 데이터를 조회하는 화면을 만듭니다.

## 요구사항

- Authorization header를 구성합니다.
- `/api/me` 또는 `/api/conversations`를 호출합니다.
- 응답 데이터를 표 또는 채팅 메시지로 표시합니다.
- 인증 실패 상황을 처리합니다.
- 04 단원의 대화 이력 미리보기와 달리, 이 과제에서는 백엔드 API 응답을 화면에 표시합니다.
- Supabase 데이터는 프론트엔드에서 직접 조회하지 않고 백엔드 API를 통해 조회합니다.
