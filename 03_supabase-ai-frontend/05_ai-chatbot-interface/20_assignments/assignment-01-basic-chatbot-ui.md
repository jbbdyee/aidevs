# Assignment 01: Basic Chatbot UI

## 목표

Streamlit 기본 채팅 컴포넌트를 사용해 챗봇 화면을 만듭니다.

## 요구사항

- `st.chat_input`으로 질문을 입력받습니다.
- `st.chat_message("user")`로 사용자 메시지를 출력합니다.
- `st.chat_message("assistant")`로 응답 메시지를 출력합니다.
- 빈 질문이 들어오지 않도록 화면 흐름을 구성합니다.

## 제출 파일 예시

```text
assignment-01-basic-chatbot-ui.py
```
