# Lab 02 Solution Notes

## 원인

샘플 백엔드는 `http://127.0.0.1:8000`에서 실행되지만 예제 코드는 `http://127.0.0.1:9999`를 호출합니다.

## 수정 방향

```python
API_BASE_URL = "http://127.0.0.1:8000"
```

## 확인

1. 백엔드를 먼저 실행합니다.
2. Streamlit 앱에서 `백엔드 상태 확인` 버튼을 누릅니다.
3. `{"status": "ok"}` 응답이 보이면 성공입니다.

## 추가 개선

- `API_BASE_URL`을 `.env` 또는 화면 입력값으로 분리할 수 있습니다.
- 실패 시 "백엔드 실행 여부와 포트 번호를 확인하세요" 같은 안내를 추가할 수 있습니다.
