# routers

이 폴더에는 FastAPI router 파일을 추가합니다.

구현할 파일 예시:

```text
log_router.py
feedback_router.py
stream_router.py
```

구현할 endpoint 예시:

```text
POST /logs
GET /logs
GET /logs/summary
GET /stream/logs
POST /feedback
GET /feedback
```

`01_realtime-log-dashboard-practice/backend/app/routers/log_router.py`를 참고하되, 팀 프로젝트 요구사항에 맞게 직접 구현합니다.
