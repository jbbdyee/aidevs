# 10_labs

`02_prompt-and-response-quality` 단원의 실습 안내입니다.

| 파일 | 주제 |
| --- | --- |
| `lab-01_improve-basic-prompt.md` | 기본 프롬프트 개선 |
| `lab-02_json-output-check.md` | JSON 출력 검증 |
| `lab-03_prompt-safety-test.md` | Prompt Injection 방어 테스트 |

## 진행 순서

```text
Lab 01 -> 프롬프트 구조 개선
Lab 02 -> 응답을 JSON/Pydantic으로 검증
Lab 03 -> 위험 입력 방어와 프롬프트 평가
```

이 단원의 lab은 “좋은 답변이 나왔다”에서 끝나지 않습니다. 응답이 일관적인지, 프로그램에서 처리 가능한지, 위험 입력에 흔들리지 않는지까지 확인합니다.
