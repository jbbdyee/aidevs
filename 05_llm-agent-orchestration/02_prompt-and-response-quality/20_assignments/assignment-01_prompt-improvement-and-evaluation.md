# Assignment 01. Prompt Improvement and Evaluation

## 과제 목표

## 작성 항목

1. 업무 시나리오
2. 개선 전 프롬프트
3. 개선 후 프롬프트
4. Structured Output 적용 여부
5. 안전성 고려 사항
6. 응답 품질 비교
