# 01_langchain-overview

이 챕터에서는 직접 API를 호출하는 방식과 LangChain을 사용하는 방식을 비교합니다.

## 핵심 개념

- LangChain은 프롬프트, 모델, 파서를 연결하는 도구입니다.
- 간단한 예제에서는 직접 호출이 더 단순할 수 있습니다.
- 흐름이 복잡해질수록 Chain 구조가 도움이 됩니다.
