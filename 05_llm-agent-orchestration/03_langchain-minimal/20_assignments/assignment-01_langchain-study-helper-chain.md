# Assignment 01. LangChain Study Helper Chain

## 과제 목표

학습 질문을 받아 학습 계획을 추천하는 간단한 LangChain Chain을 설계합니다.

## 요구 사항

1. 입력: 학습 질문 또는 학습 주제
2. 처리: PromptTemplate 또는 ChatPromptTemplate 사용
3. 출력: 단계별 학습 계획
4. 가능하면 Pydantic Structured Output 적용

## 제출 기준

- Chain 구조 설명
- 실행 코드 또는 의사코드
- 예시 입력과 출력
- 개선할 점
