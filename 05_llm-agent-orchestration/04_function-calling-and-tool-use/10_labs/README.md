# 10_labs

`04_function-calling-and-tool-use` 단원의 실습 안내입니다.

| 파일 | 주제 |
| --- | --- |
| `lab-01_calculator-tool.md` | 계산 Tool 구현 |
| `lab-02_learning-log-tool.md` | 학습 로그 요약 Tool 구현 |
| `lab-03_multi-tool-selection.md` | Multi-Tool 선택 흐름 설계 |

## 진행 순서

```text
Lab 01 -> 가장 단순한 계산 Tool 구조 이해
Lab 02 -> 업무 데이터 조회 Tool 구조 이해
Lab 03 -> 여러 Tool 중 필요한 Tool을 선택하는 기준 설계
```

이 단원의 lab은 “함수가 실행된다”에서 끝나지 않습니다. Tool 이름, 입력값, 출력값, 실패했을 때의 응답까지 함께 정리합니다.
