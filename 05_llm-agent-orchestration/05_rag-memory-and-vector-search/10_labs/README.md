# 10 Labs

이 폴더는 `05_rag-memory-and-vector-search` 단원의 실습 안내입니다. 실습은 API 없이 이해할 수 있는 개념 예제에서 시작해 pgvector, RAG, Memory, 품질 평가 순서로 진행합니다.

| 파일 | 주제 | 확인할 내용 |
| --- | --- | --- |
| `lab-01_embedding-similarity.md` | Embedding 유사도 | 문장 의미를 벡터 거리로 비교하는 흐름 |
| `lab-02_pgvector-search.md` | pgvector 검색 | Docker DB에 벡터를 저장하고 검색하는 과정 |
| `lab-03_basic-rag-answer.md` | Basic RAG 답변 | 검색 context를 LLM 답변에 연결하는 구조 |
| `lab-04_session-memory.md` | Session Memory | 현재 대화 상태를 유지하는 방법 |
| `lab-05_hybrid-search-rrf.md` | Hybrid Search와 RRF | 키워드/벡터 검색 결과 결합 방식 |
| `lab-06_rag-quality-evaluation.md` | RAG 품질 평가 | 근거성, 관련성, 일관성 점검 |

## 실습 기록 기준

- 실행한 명령어
- 입력 질문 또는 샘플 문서
- 검색된 context
- 생성된 답변
- 실습 중 발생한 오류와 해결 방법
