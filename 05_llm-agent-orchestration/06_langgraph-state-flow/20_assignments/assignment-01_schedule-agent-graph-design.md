# Assignment 01. Schedule Agent Graph Design

## 과제 목표

일정 조정 Agent를 LangGraph StateGraph로 설계합니다.

## 작성 항목

1. State 필드
2. Node 목록
3. Edge 흐름
4. 조건 분기 기준
5. Tool 호출 위치
6. Memory 사용 여부
7. Retry와 Fallback 전략

## 평가 기준

- State 설계가 명확한가
- Node 책임이 분리되어 있는가
- 조건 분기와 실패 대응이 포함되어 있는가
- 최종 프로젝트로 확장 가능한가
