# Tool Spec

이 문서는 샘플 일정 조정 Agent가 사용하는 Tool의 입력과 출력을 정리합니다.

## Tool 목록

| Tool | 입력 | 출력 | 역할 |
| --- | --- | --- | --- |
| `check_calendar_tool` | 참석자 목록 | 참석자별 일정 | Mock 일정 조회 |
| `find_available_slot_tool` | 일정, 회의 시간 | 가능한 시간 후보 | 일정 조정 |
| `draft_schedule_message_tool` | 선택 시간, 참석자 | 제안 메시지 | 사용자 응답 생성 |

## 설계 기준

- Tool은 하나의 책임만 갖습니다.
- 입력과 출력은 명확한 dict 또는 list 구조로 만듭니다.
- 실패할 수 있는 상황을 미리 정리합니다.

## 실패 상황 예시

| 상황 | 처리 |
| --- | --- |
| 참석자 정보 없음 | 사용자에게 추가 정보를 요청 |
| 가능한 시간 없음 | 대체 날짜를 제안 |
| Tool 결과 비어 있음 | Fallback 메시지 생성 |
