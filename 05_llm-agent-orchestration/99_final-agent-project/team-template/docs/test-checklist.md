# Test Checklist

최종 Agent 미니 프로젝트에서 확인할 테스트 체크리스트입니다.

## 실행 준비

- [ ] `.venv`를 만들었습니다.
- [ ] `.venv`를 활성화했습니다.
- [ ] `python -m pip install --upgrade pip`를 실행했습니다.
- [ ] `pip install -r .\backend\requirements.txt`를 실행했습니다.
- [ ] 필요한 `.env` 값을 설정했습니다.

## 기능 테스트

- [ ] CLI 실행이 성공합니다.
- [ ] Streamlit 화면이 실행됩니다.
- [ ] 사용자 입력을 받을 수 있습니다.
- [ ] Tool 함수가 2개 이상 동작합니다.
- [ ] State가 단계별로 업데이트됩니다.
- [ ] 최종 답변이 표시됩니다.

## Agent 구조 테스트

- [ ] State 필드가 명확합니다.
- [ ] Node 책임이 분리되어 있습니다.
- [ ] Edge 흐름이 문서화되어 있습니다.
- [ ] 조건 분기 또는 Retry 흐름이 있습니다.
- [ ] Fallback 메시지가 있습니다.

## 발표 전 확인

- [ ] 프로젝트 목적을 설명할 수 있습니다.
- [ ] Tool 설계 이유를 설명할 수 있습니다.
- [ ] 실패 상황 대응을 설명할 수 있습니다.
- [ ] 개선할 점을 정리했습니다.
