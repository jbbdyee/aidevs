# 01_project-deliverables

이 폴더는 `06_llm-agent-mini-project`의 필수 산출물 샘플입니다.

주제:

```text
복합 API 연계 일정 조정 에이전트
```

## 필수 산출물

| 파일 | 산출물 |
| --- | --- |
| `01_agent-architecture-design-sample.md` | 에이전트 아키텍처 설계서 |
| `02_agent-test-result-report-sample.md` | 에이전트 시험 결과 보고서 |

## 선택 산출물

| 산출물 | 설명 |
| --- | --- |
| 발표 자료 | 최종 발표가 필요한 경우 작성합니다. |
| 배포 노트 | Render, Streamlit Community Cloud 등 선택 배포를 진행한 경우 작성합니다. |
| 회고 문서 | 팀 역할, 막힌 지점, 개선 방향을 정리합니다. |

## 작성 기준

샘플 문서는 그대로 제출하는 문서가 아닙니다. 팀 프로젝트의 실제 구현 내용, 테스트 결과, 오류 수정 이력을 반영해 수정해야 합니다.

## 제출 전 확인

- [ ] StateGraph Node와 Edge가 실제 코드와 일치한다.
- [ ] Tool 목록과 입력/출력이 실제 코드와 일치한다.
- [ ] 테스트 결과 보고서에 성공 사례뿐 아니라 실패와 수정 과정도 포함되어 있다.
- [ ] Self-Reflection 적용 전후 비교가 있다.
- [ ] `.env`, API Key, 개인정보가 문서에 들어가지 않았다.
