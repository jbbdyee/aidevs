# 10 Labs

이 폴더는 Multi-Agent 협업 설계를 실습하는 lab 문서를 모아둔 곳입니다.

## Lab 목록

| Lab | 내용 |
| --- | --- |
| `lab-01_single-vs-multi-agent.md` | 단일 Agent와 Multi-Agent 비교 |
| `lab-02_role-based-agent-design.md` | 역할 기반 Agent 설계 |
| `lab-03_supervisor-routing.md` | Supervisor/Router 작업 분배 |
| `lab-04_distributed-collaboration.md` | 분산 협업과 결과 통합 |
| `lab-05_handoff-context-mcp.md` | Handoff, Context 공유, MCP 개념 |
| `lab-06_feedback-loop-result-review.md` | Feedback Loop와 결과 검증 |

## 공통 제출 기준

- 실행 명령과 출력 결과를 기록합니다.
- 어떤 Agent가 어떤 책임을 가지는지 정리합니다.
- Agent 간 전달되는 Context 필드를 정리합니다.
- 실패 상황에서 retry 또는 fallback 기준을 적습니다.
