# 20 Assignments

## 과제. Multi-Agent 협업 설계서 작성

아래 주제 중 하나를 선택해 Multi-Agent 협업 설계서를 작성합니다.

## 선택 주제

1. 장애 자동 진단 Agent
2. 서비스 로그 분석 Agent
3. 배포 전 검증 Agent
4. 고객 문의 분류 Agent

## 제출 내용

- 문제 정의
- Agent 역할 목록
- Supervisor/Router 규칙
- Handoff Context 구조
- Agent 간 Context 동기화 기준
- Feedback Loop 기준
- 실패 시 fallback 전략

## 평가 기준

- Agent 역할이 명확한가?
- Context 전달 필드가 구체적인가?
- 결과 검증과 재시도 기준이 있는가?
- 이후 Docker Compose 서비스 구조로 확장할 수 있는가?
