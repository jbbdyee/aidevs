# 10 Labs

이 폴더는 Docker, Docker Compose, GitHub Actions, AWS 배포, AWS 리소스 삭제를 단계별로 실습하는 lab 문서입니다.

## Lab 목록

| Lab | 내용 |
| --- | --- |
| `lab-01_docker-build-and-run.md` | 단일 FastAPI 서비스를 Docker build/run 합니다. |
| `lab-02_docker-compose-up.md` | Docker Compose로 멀티 서비스를 실행합니다. |
| `lab-03_health-check-and-logs.md` | Health Check와 로그를 확인합니다. |
| `lab-04_github-actions-docker-build.md` | GitHub Actions Docker build 검증을 확인합니다. |
| `lab-05_aws-deployment-checklist.md` | AWS 배포 전 준비 상태를 점검합니다. |
| `lab-06_aws-apprunner-deployment.md` | ECR image push와 App Runner 배포를 진행합니다. |
| `lab-07_aws-cleanup-and-cost-control.md` | AWS 리소스 삭제와 비용 확인을 진행합니다. |

## 공통 제출 기준

- 실행한 명령
- 실행 결과 화면 또는 출력
- 오류가 있었다면 해결 과정
- 운영 관점에서 확인한 내용
- AWS 실습의 경우 배포 URL, CloudWatch Log Group, 삭제 확인 결과
