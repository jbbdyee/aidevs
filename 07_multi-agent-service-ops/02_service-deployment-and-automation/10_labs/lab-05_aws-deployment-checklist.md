# Lab 05. AWS Deployment Checklist

## 목표

AWS 배포 전 필요한 항목을 체크리스트로 정리합니다.

## 작성할 내용

- 사용할 배포 방식: App Runner 또는 ECS
- Docker image 이름
- ECR repository 이름
- container port
- health check path
- 필요한 환경변수
- 로그 확인 위치
- 실습 후 삭제할 리소스

## 주의

AWS 실습은 비용이 발생할 수 있습니다. 실제 배포는 필수로 진행하되, 배포 전 Budget을 설정하고 실습 후 리소스를 삭제합니다.
