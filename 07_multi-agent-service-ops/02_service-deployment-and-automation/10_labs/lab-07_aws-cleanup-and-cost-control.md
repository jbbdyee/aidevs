# Lab 07. AWS Cleanup And Cost Control

이 lab은 AWS 배포 실습 후 리소스를 삭제하고 비용을 확인하는 절차입니다.

## 목표

- App Runner service를 삭제합니다.
- ECR image/repository를 삭제합니다.
- CloudWatch Log Group을 확인하고 삭제합니다.
- Billing Dashboard에서 비용을 확인합니다.

## 진행 순서

1. App Runner service 삭제
2. ECR repository 삭제
3. CloudWatch Log Group 삭제
