# Assignment 01. Docker Compose Service Design

## 과제 목표

팀 미니 프로젝트를 Docker Compose 기반 서비스 구조로 설계합니다.

## 필수 작성 내용

1. 서비스 목록
2. 각 서비스 역할
3. 각 서비스 포트
4. 필요한 환경 변수
5. Health Check 방식
6. 로그 확인 방법
7. AWS 배포 시 사용할 서비스 후보

## 예시 서비스

```text
backend: FastAPI API 서버
frontend: Streamlit 화면
worker: Agent 작업 실행기
monitor: 운영 상태 대시보드
```
