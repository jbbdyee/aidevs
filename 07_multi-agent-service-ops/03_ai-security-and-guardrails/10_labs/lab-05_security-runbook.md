# Lab 05. Security Runbook

## 목표

AI 서비스 보안 운영 가이드라인을 작성합니다.

## 포함할 내용

- Prompt Injection 의심 입력 처리 절차
- 정책 위반 응답 발생 시 처리 절차
- Tool 권한 위반 발생 시 처리 절차
- 민감 정보 노출 의심 시 처리 절차
- 로그 확인 위치
- 조치 결과 기록 방식

## 작성 예시

```text
1. 위반 이벤트 확인
2. request_id 기준 로그 조회
3. 입력/응답/Tool 호출 이력 확인
4. 민감 정보 포함 여부 확인
5. 차단 또는 재실행 결정
6. 조치 결과 기록
```
