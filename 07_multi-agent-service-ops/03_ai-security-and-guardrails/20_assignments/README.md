# 20 Assignments

## 과제. AI 보안 및 Guardrails 통합 설계서

Multi-Agent 서비스에 적용할 보안/가드레일 설계서를 작성합니다.

## 제출 내용

- Prompt Injection 방어 기준
- 입력 검증 기준
- 응답 정책 검증 기준
- Tool 권한 제어 표
- Multi-Agent 접근 제어 표
- 감사 로그 필드 설계
- 정책 위반 발생 시 운영 절차
- Guardrails AI 같은 검증 도구를 붙일 위치

## 평가 기준

- 입력, Tool 실행, 응답 검증 흐름이 분리되어 있는가?
- Agent 역할별 권한이 명확한가?
- 정책 위반과 권한 위반을 추적할 수 있는가?
- 운영 중 사람이 확인할 수 있는 로그 구조가 있는가?
