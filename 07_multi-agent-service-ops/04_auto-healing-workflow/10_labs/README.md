# 10 Labs

이 폴더는 Auto Healing 흐름을 단계별로 실습하는 lab 문서입니다.

## Lab 목록

| Lab | 내용 |
| --- | --- |
| `lab-01_failure-scenario-classification.md` | 장애 유형 분류 |
| `lab-02_health-check-retry-restart.md` | Health Check, Retry, Restart |
| `lab-03_auto-healing-pipeline.md` | 복구 파이프라인 설계 |
| `lab-04_recovery-result-validation.md` | 복구 결과 검증 |

## 공통 기준

- 장애 유형을 명확히 적습니다.
- 선택한 복구 전략의 이유를 적습니다.
- 복구 전후 상태를 비교합니다.
- 이벤트 로그에 남길 정보를 정리합니다.
