# 20 Assignments

## 과제. Auto Healing 장애 대응 설계서

AI 서비스 운영 중 발생할 수 있는 장애를 하나 선택하고 Auto Healing 설계서를 작성합니다.

## 제출 내용

- 장애 시나리오
- 장애 감지 기준
- 장애 유형 분류
- retry/restart/fallback 전략
- 최대 재시도 횟수
- 복구 결과 검증 기준
- 이벤트 로그 필드
- 수동 확인이 필요한 조건

## 평가 기준

- 장애 유형과 복구 전략이 연결되어 있는가?
- 무한 재시도를 방지하는가?
- 복구 후 검증 기준이 있는가?
- 운영자가 확인할 로그가 남는가?
