# CloudWatch Log Checklist

AWS 배포 후 CloudWatch Logs 확인 결과를 기록합니다.

## 체크리스트

- [ ] CloudWatch Log Groups 화면을 열었다.
- [ ] App Runner service와 관련된 Log Group을 찾았다.
- [ ] 최신 Log Stream을 열었다.
- [ ] 배포 성공 또는 실패 로그를 확인했다.
- [ ] 애플리케이션 startup 로그를 확인했다.
- [ ] `/health` 요청 로그를 확인했다.
- [ ] 오류 로그가 있다면 원인을 정리했다.
- [ ] 로그에 API Key, Access Key, 개인정보가 노출되지 않았다.
