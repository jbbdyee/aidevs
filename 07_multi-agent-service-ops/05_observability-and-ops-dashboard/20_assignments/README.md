# 20 Assignments

## 과제. 운영 대시보드 설계서

Multi-Agent 서비스의 운영 상태를 확인할 수 있는 대시보드 설계서를 작성합니다.

## 제출 내용

- 기록할 이벤트 로그 필드
- trace 구조
- 실행 상태 전이표
- 대시보드에 표시할 지표
- 실패 이벤트 표시 방식
- Auto Healing 결과 표시 방식
- LangSmith 같은 외부 추적 도구를 붙일 위치

## 평가 기준

- request_id 기준으로 실행 이력을 추적할 수 있는가?
- 실패 단계와 복구 결과를 확인할 수 있는가?
- 운영자가 다음 행동을 판단할 수 있는 정보가 있는가?
