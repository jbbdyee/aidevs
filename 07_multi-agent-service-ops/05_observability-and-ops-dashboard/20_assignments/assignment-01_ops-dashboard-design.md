# Assignment 01. Ops Dashboard Design

## 과제 목표

팀 미니 프로젝트에 필요한 운영 관찰 구조와 대시보드를 설계합니다.

## 필수 작성 내용

1. 수집할 로그 필드 목록
2. Agent 실행 Trace 단계 목록
3. 서비스별 Health Check 정보
4. 운영 대시보드 화면 구성
5. 실패 이벤트와 복구 이벤트 표시 방식
6. Docker Compose monitor 서비스 구성 아이디어
7. AWS 운영 환경에서 연결할 서비스 후보

## 제출 형식

Markdown 문서로 제출합니다.

```text
team-name_ops-dashboard-design.md
```

## 평가 기준

- 운영자가 문제를 빠르게 파악할 수 있는 정보가 있는가?
- 로그, Trace, 상태 관리가 구분되어 있는가?
- Auto Healing 결과를 확인할 수 있는가?
- Docker/AWS 확장 관점이 포함되어 있는가?
