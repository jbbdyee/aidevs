# 01_project-deliverables

이 폴더는 08 미니 프로젝트의 필수 산출물 샘플입니다. 팀 프로젝트를 진행할 때는 아래 3개 문서를 반드시 작성합니다.

## 필수 산출물

| 산출물 | 샘플 파일 |
| --- | --- |
| 멀티 에이전트 아키텍처 설계서 | `01_multi-agent-architecture-design-sample.md` |
| 배포 및 장애 복구 보고서 | `02_deployment-and-recovery-report-sample.md` |
| 파이프라인 구현 결과 보고서 | `03_pipeline-implementation-result-report-sample.md` |

## 작성 원칙

- 코드 설명만 쓰지 말고, 왜 이런 구조를 선택했는지 근거를 씁니다.
- 실제 실행 결과, 캡처, 로그, URL, 실패 사례를 함께 정리합니다.
- `.env`, API Key, AWS Access Key, 비밀번호는 절대 문서에 적지 않습니다.
- 구현하지 못한 항목은 “미구현”으로 표시하고 이유와 보완 계획을 적습니다.
